# Dropbox GSAP Clone

Clone da Homepage do [Dropbox](https://www.dropbox.com/pt_BR/). O objetivo do projeto é praticar o desenvolvimento de animações mais complexas usando GSAP e ScrollTrigger.

Você pode acessar o projeto aqui: [Dropbox GSAP Clone](https://givanyo.github.io/dropbox-gsap-clone/)

## Tecnologias usadas

- HTML5
- CSS
- JavaScript
- GSAP
- ScrollTrigger
